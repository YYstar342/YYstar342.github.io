7.JavaDoc和API

API:

​ 应用程序编程接口，是一组定义和规则，允许不同的软件或系统之间进行通信。它提供了一种标准方式，使开发者能够使用特定的功能，而无需了解其底层实现。

JavaDoc:

​ 是 Java 编程语言中用于生成 API 文档的工具。它通过解析 Java 代码中的特殊注释（

注释以 /** 开头，以 */ 结尾，通常写在类、方法、字段等声明之前。注释内容可以包含 HTML 标签和 Javadoc 标签（以 @ 开头的特殊标记）），自动生成 HTML 格式的文档，方便开发者查阅类、方法和字段的详细信息。

常见标签：

标签 作用
@author 说明代码作者
@version 说明版本号
@param 说明方法的参数
@return 说明方法的返回值
@throws 说明方法可能抛出的异常
@see 参考相关类或方法
@deprecated 标记已废弃的方法或类

4.中断控制流程的语句

Break:

​ 是 Java 中的一个控制流语句，用于 立即终止循环 或 跳出 switch 语句。它的作用是提前结束当前代码块的执行，并跳转到代码块之后的语句。

break 的两种主要用途

  1. 在循环中使用 break：
    • 用于提前终止循环（如 for、while、do-while）。
    • 当满足某个条件时，可以使用 break 立即退出循环，不再执行循环中剩余的代码。
  2. 在 switch 语句中使用 break：
    • 用于跳出 switch 语句，防止“贯穿”（fall-through）现象。
    • 如果没有 break，switch 语句会继续执行后续的 case 或 default 块。
